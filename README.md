# 🚀 Stellar Wallet App (Level 1)

## 📌 Project Description
Stellar Wallet App is a blockchain-based web application built using React, Vite, JavaScript, and Stellar SDK.

The application allows users to:
- Create a new Stellar wallet
- Generate Public Key and Secret Key
- Fund wallet using Friendbot on Testnet
- Check wallet balance
- Send XLM to another wallet
- View transaction status

This project demonstrates blockchain wallet management, secure transactions, and real-time balance tracking using the Stellar network.

---

## ✨ Features

- Wallet creation
- Public / Secret key generation
- Testnet funding with Friendbot
- Real-time balance checking
- Send XLM payments
- Success / Failure transaction alerts
