# 🚀 Stellar Wallet App (Level 1)

## 📌 Project Description
Stellar Wallet App is a blockchain-based web application built using React, Vite, JavaScript, and Stellar SDK.

It allows users to:
- Create a Stellar wallet
- Generate Public and Secret Keys
- Fund account using Friendbot
- Check wallet balance
- Send XLM tokens
- View transaction status

This project demonstrates wallet management, blockchain payments, and real-time balance tracking on Stellar Testnet.
