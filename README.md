# 🚀 Stellar Wallet App (Level 1)

## 📌 Description
A basic Stellar blockchain wallet app built with React and Stellar SDK.

## ✨ Features
- Create wallet
- Generate Public/Secret key
- Fund wallet with Friendbot
- Check balance
- Send XLM
- Show transaction result

## ⚙️ Run Project
npm install
npm run dev
